# Name

EARTH Tiles

# URL

opensea/earth-tiles

# Description

`EARTH` defines an equal area mesh for planet Earth. The mesh consists of 812 tiles, out of which 12 are **pentagons** and the rest are **hexagons**.
Tile owners can set a **custom message** for the world to see.
